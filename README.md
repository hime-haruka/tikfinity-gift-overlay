# 🎁 TikFinity 기프트 오버레이 사용 안내

안녕하세요!  
이 오버레이는 TikTok LIVE 후원을 화면에 표시해주는 기능입니다.

아래 순서대로 진행해주시면 바로 사용하실 수 있습니다.

---

## 🧩 1. 사전 준비

먼저 아래 프로그램을 실행해주세요.

1. TikFinity Desktop App 실행  
2. TikTok LIVE 방송 연결

👉 TikFinity가 실행되어 있어야 후원 이벤트를 받을 수 있습니다.

---

## 🚀 2. Receiver 프로그램 실행

제공받은 프로그램을 실행해주세요.

TikFinity Gift Receiver.exe

실행 후 아래 정보를 입력합니다.

Client ID: (제공받은 값)  
Server URL: (제공받은 주소)

입력 후 **Start 버튼을 클릭**해주세요.

---

## ✅ 3. 연결 상태 확인

정상적으로 연결되면:

- TikFinity 상태 → Connected (초록색)
- Server 상태 → Connected (초록색)

으로 표시됩니다.

연결이 되지 않을 경우:

- TikFinity 실행 여부 확인
- 방송 연결 여부 확인
- Server URL 오타 확인

---

## 🎥 4. OBS에 오버레이 추가

Receiver 프로그램에서 아래 버튼을 눌러주세요.

[Overlay URL 복사]

이후 OBS에서:

1. 소스 추가 → 브라우저
2. URL에 붙여넣기
3. 해상도 설정 (권장: 1920x1080)

---

## ⚙️ 5. 설정 변경

아래 페이지에서 오버레이 설정을 변경할 수 있습니다.

[설정 페이지 URL 복사] 버튼 클릭

설정 가능한 항목:

- 기프트 이름 표시 ON/OFF
- 기프트 이미지 표시 ON/OFF
- 후원자 프로필 표시 ON/OFF
- 다이아 표시 여부
- 최소 표시 금액
- 정렬 방식 (최신순 / 금액순)
- 카드 색상 (텍스트 / 배경 / 테두리 / 그라데이션)
- 레벨업 카드 색상 별도 설정

---

## 🧠 참고 사항

- 기프트는 설정한 최소 금액 이상만 표시됩니다.
- 멤버 레벨업은 실제로 레벨이 상승했을 때만 표시됩니다.
- 프로그램을 껐다 켜도 설정은 유지됩니다.
- Client ID는 다른 사용자와 공유하지 마세요.

---

## ⚠️ 문제 발생 시

아래 항목을 확인해주세요:

1. TikFinity 실행 여부  
2. 방송 연결 여부  
3. Receiver 실행 여부  
4. Client ID / Server URL 입력값 확인

문제가 계속될 경우 제작자에게 문의해주세요.
