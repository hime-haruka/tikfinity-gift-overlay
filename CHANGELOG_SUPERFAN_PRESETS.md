# 추가 수정 내역

## 슈퍼팬 처리
- TikTok-Live-Connector의 `superFan`, `superFanJoin`, `superFanBox` 이벤트를 감지합니다.
- 이벤트가 들어오면 해당 유저를 `client.superFans`에 저장합니다.
- 저장 대상에 `superFans`를 추가해서 서버 재시작 후에도 유지됩니다.
- 이후 같은 유저가 기프트를 보내면 `isSuperFan: true`가 붙고 슈퍼팬 컬러가 적용됩니다.
- 수동 입력형 `슈퍼 팬 ID/닉네임 목록` UI는 제거했습니다.

## 컬러 프리셋 UI
- 드롭다운 프리셋 선택을 제거하고 원형 그라데이션 미리보기 버튼으로 변경했습니다.
- 선택 중인 프리셋에는 하이라이트와 `선택중` 배지가 표시됩니다.
- 프리셋 클릭 시 기본 기프트/레벨업 컬러, 슈퍼팬 컬러, 금액별 컬러, 레벨별 컬러가 함께 적용됩니다.

## 프리셋 추가
- 코튼 캔디
- 스카이 클라우드
- 민트 소다
- 퍼플 드림
- 피치 크림
- 로즈 골드
- 프리미엄 골드
- 네온 나이트
- 모노크롬

## 테스트 기능
- URL/테스트 탭에 `테스트 슈퍼팬 등록` 버튼을 추가했습니다.
- 테스트 슈퍼팬 등록 후 같은 테스트 유저가 기프트를 보내도록 해서 슈퍼팬 컬러 적용을 바로 확인할 수 있습니다.
